Prepares you for ePrivacy

The "Opt-In Layer" is a JavaScript library for displaying the user consent dialog. It offers a huge variety of options and helps you to manage your tags.
Best served with Tealium!

We can help you. No worries.

We help you with our battle proven and ePrivacy approved solution. You don't know what's required for ePrivacy? No problem. We do.

We looked at every known JS-Tag and categorized them. With the help of any tag manager using OIL.js becomes as easy as adding the library.

Some of our features

  • Direct or complete integration with tealium
  • Single Opt-In for one website
  • Power Opt-In with third party cookie softblock fallback
  • Power Opt-In for groups of websites (1-n POI-groups)
  • Central domain oil.axelspringer.com as cookie registry for Power Opt-In
  • Many demos, examples, and documentation
  • Domain White Listing against unauthorized usage
  • Cookie Preference Center (embeddable)
  • Highly Customizable Layer - just some lines of CSS away
  • Multiple possible buttons ("Yes", "Later", "CPC", "Ignore", "X")
  • All texts are configurable
  • Javascript events for everything
  • Preview and debug modes
  • Works with all resolutions and device types

OIL.JS QUICK INSTALL

You have two options: Including the JS-Library via our CDN or deploying it with Tealium! For Tealium you only need to include our dip-lib-opt-in profile and you're ready to go. For direct integration you only need to add our tag: 

# This will add the layer with default options and default styling to your website

<script type="text/javascript" src="//oil.axelspringer.com/release/oil.1.0.13-RELEASE.min.js"></script>

And if you want some options:

<script id="oil-configuration" type="application/configuration">
    {
        "privacy_page_url": "http://www.mysite.com/privacy",
        "label_intro_heading": "- You name it! -",
        "poi_activate_poi": true,
        "poi_hub_origin": "//oil.axelspringer.com",
        "poi_hub_path": "/hub.html",
    }
</script>
